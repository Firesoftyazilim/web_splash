# Flutter Web Splash

Bu kütüphane Flutter web de animasyonlu bir açılış sayfası eklemek için yapılmıştır.

![](https://github.com/Firesoftyazilim/web_splash/blob/main/web_splash.gif)




## Kurulum

Terminalde çalıştırın.

```bash
flutter pub add il_ilce
```

## Kullanım
Terminalde çalıştırın.

```bash
dart web_splash
```
